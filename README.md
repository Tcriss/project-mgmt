## Description

This is an api to test my knowledge made with NestJS, one of the best frameworks nodeJS have.

It's for projects organization and management. You can create projects, tasks, and handle them through a user.

## Installation

```bash
$ npm install
```

## Building the database from docker

```bash
$ docker compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
